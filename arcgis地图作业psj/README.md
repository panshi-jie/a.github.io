# 智慧城市时空信息平台

基于 ArcGIS Maps SDK for JavaScript 5.0 开发的交互式地图应用。

## 功能特性

- 🗺️ **多底图支持**：右侧底图库提供多种底图样式，一键切换
- 📍 **专题图层**：展示北京地区兴趣点分布和世界城市数据
- 🔍 **智能搜索**：支持地点搜索和定位
- 📏 **比例尺**：动态显示当前地图比例
- 📖 **图例**：清晰展示图层符号含义
- 📋 **图层控制**：可控制各图层的显示/隐藏和透明度
- 🎨 **美观界面**：现代化渐变页眉、浮动面板设计

## 技术栈

- ArcGIS Maps SDK for JavaScript 5.0
- HTML5/CSS3
- JavaScript ES6
- Flexbox/Grid 响应式布局

## 快速开始

1. 克隆项目到本地
2. 使用任意 HTTP 服务器运行（如 Live Server）
3. 确保网络可以访问 ArcGIS CDN 服务
